Single User Guide Display Template Help

Assigned Variables

  • item (object) empty or populated item being added/edited
  • newitem (bool) true when creating a new guide
  • input_content (html elements) for WYSIWYG editing of item content, if that's wanted
  • sheets_list (array) numeric id(s) and name(s) of stylesheets any of which can be selected for application to the item
  • templates_list (array) numeric id(s) and name(s) of templates one of which can be selected for application to the item<
  • tab (string) identifier of the active tab, possibly empty

Properties of Each Item

  • active (bool) whether the guide is enabled/active
  • admin (bool) whether the guide is for display only in the admin-console
  • author (string) identifier of the creator or some later modifier
  • author_id (int) numeric id of the latest modifier
  • content (string) what is displayed
  • create_date (string) item creation date-time, formatted as in the database, like Y-m-D H:i:s
  • id (int) numeric identifier 0 for new item, > 0 for existing
  • name (string) displayed identifier
  • modified_date (string) date-time of latest modification if any, formatted as in the database, like Y-m-D H:i:s
  • position (int) list display sort-order index
  • revision (string) suppementary identifier, for version control etc
  • restricted (int) type of restriction, if any, that applies to displaying the guide in the Admin Console
  • search (bool) whether the name and content can be accessed by the AdminSearch module
  • sheets (string) comma-separated numeric id(s) of styleheet(s) to use for frontend display
  • smarty (bool) whether to process the content via Smarty before display
  • template_id (int)
  • wysiwyg (bool) whether the item normally is edited using the WYSIWYG editor
