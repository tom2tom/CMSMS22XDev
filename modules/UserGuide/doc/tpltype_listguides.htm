User Guides List Template Help

Assigned Variables

  • mod (UserGuide object) module
  • guides (array of arrays) items to be listed
  • iconurl (string) website URL of custom 'view' icon

Potentially Relevant Members of Each Item-Array

  • id (int) item's numeric identifier > 0
  • name (string) recognisable identifier
  • revision (string) suppementary identifier, for version control etc
  • author (string) identifier of the item's creator, or some later modifier
  • create_date (string) date-time of item's creation, formatted like Y-m-D H:i:s
  • modified_date (string) date-time of item's latest modification if any, formatted like Y-m-D H:i:s
  • latest (string) latter of modified_date, create_date
